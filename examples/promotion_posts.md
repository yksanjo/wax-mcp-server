## 📧 Hacker News Post

**Title:** Show HN: First MCP Server for the WAX Blockchain — AI Agents Can Now Interact with WAX

**Body:**
> I built the first MCP (Model Context Protocol) server for the WAX blockchain. This lets AI agents like Claude, Cursor, and Cline query on-chain data through natural language.
>
> **Tools available:**
> • wax_get_balance — Check WAXP and token balances
> • wax_get_nfts — Query NFT assets by owner or collection
> • wax_get_collection — Get detailed collection info
> • wax_list_collections — Browse top NFT collections
> • wax_get_sales — Recent marketplace sales
> • wax_get_price — Current WAXP price
> • wax_get_market_data — Market cap, volume, supply
> • wax_get_transactions — Transaction history
>
> Also built a companion Python SDK (wax-agent-toolkit) for developers who want to build custom AI agents, trading bots, or portfolio trackers on WAX.
>
> **Why WAX?** It's one of the largest NFT ecosystems by daily volume, but has almost no AI/developer tooling. This is the first MCP server for the ecosystem.
>
> **Stack:** Python, MCP protocol, WAX RPC, AtomicAssets API, CoinGecko
>
> Both are MIT licensed:
> • github.com/yksanjo/wax-mcp-server
